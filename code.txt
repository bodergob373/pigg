            <strong class = "smallHeader"> Materials: </strong>
            <li> <strong class = "listHeader"> Electric Motor: <img class = "listImage" src = "https://m.media-amazon.com/images/I/51rVY5Wx5VL._AC_SY200_QL15_.jpg" width = "100" align = "top"> </strong> <br>

            A source of electrical power. The positive and negative terminals can be connected to a circuit to create a current. </li>
            <li> <strong class = "listHeader"> Switch: <img class = "listImage" src = "https://cdn.sparkfun.com/assets/parts/5/6/1/9/10727-01.jpg" width = "100" align = "top"> </strong> <br>

            <li> <strong class = "listHeader"> Wires: </strong> <br>
            Wires allow electricity to flow through them. Use a wire stripper to remove unneeded insulation. </li>
